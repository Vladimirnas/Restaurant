Предметная область: Ресторан
Классы:
Employee (Абстрактный класс сотрудников ресторана)
Waiter (Наследник класса Employee, класс официанта)
Chef (Наследник класса Employee, класс повара)
MenuItem (Класс для описания блюд в меню)
Order (Класс для описания заказа)
Table (Класс для столиков ресторана)
Customer (Класс для клиентов ресторана)
Restaurant (Основной класс ресторана, управляющий объектами)
